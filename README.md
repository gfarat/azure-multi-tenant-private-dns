# 🌐 Azure Multi-tenant Private DNS Architecture

Real-world implementation of a **multi-tenant hybrid DNS architecture** using:

- Azure Private Endpoints  
- Azure Policy (deployIfNotExists)  
- Azure Lighthouse  
- Azure Private Resolver  
- Hybrid DNS integration
